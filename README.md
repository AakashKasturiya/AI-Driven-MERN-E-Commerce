# 📖 Overview

Velora is a modern AI-powered Ecommerce platform built using the MERN Stack.

The project demonstrates production-level frontend architecture, secure backend APIs, authentication, shopping cart management, wishlist, checkout flow, order management, and scalable component-based development.

This project is being developed as a portfolio-ready application for Product Based Companies.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/velora-ai-ecommerce.git
```

---

## Frontend

```bash
cd client
npm install
npm run dev
```

---

## Backend

```bash
cd server
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server.

```env
PORT=5000

MONGODB_URI=YOUR_DATABASE_URL

JWT_SECRET=YOUR_SECRET_KEY

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
